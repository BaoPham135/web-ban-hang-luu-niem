II. MÔ TẢ ĐỀ TÀI
----------------------------------------------------------------------
Tên đề tài: nghiên cứu và xây dựng website bán hàng lưu niệm

Mô tả ngắn:
Đề tài tập trung nghiên cứu và xây dựng website bán hàng lưu niệm với các chức năng như xem sản phẩm, giỏ hàng và đặt mua trực tuyến.
Hệ thống giúp cửa hàng quản lý sản phẩm hiệu quả và mang đến trải nghiệm mua sắm thuận tiện cho khách hàng.
----------------------------------------------------------------------
III. CÁCH CÀI ĐẶT & CHẠY DỰ ÁN (LOCALHOST - XAMPP)
----------------------------------------------------------------------
1. Cài đặt XAMPP
2. Copy toàn bộ thư mục SourceCode vào:
   htdocs/My_web/
3. Khởi động Apache và MySQL
4. Import Database:
   - Mở phpMyAdmin
   - Tạo database mới: <web_mysqli>
   - Import file: Database/web_mysqli.sql
5. Chạy dự án:
   http://localhost/<My_web>
